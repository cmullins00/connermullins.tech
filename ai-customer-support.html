AI Customer
Support

I developed a customer support chatbot with the help of two other fellows. The chatbot was designed to assist in effectively navigating a given platform. This includes answering questions about the platform's features, interview process, as well as provide support for account issues and technical problems. My primary contributions includes but is not limited to: UI design, establishing the backend logic and API connection, and creating features that improved the user experience such as auto scrolling when new messages appear.

This chatbot is powered by OpenAI's ChatGPT. The chatbot has a list of requirements that it is fed to initialize it for company uses. The knowledge that the bot has is completely customizeable, allowing it to be set up for nearly any task. The app was built using explicitely JavaScript, from the UI to the backend logic.
